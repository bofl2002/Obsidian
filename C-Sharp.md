[[Programmering]]




**C#** (udtales "C-sharp") er et moderne, objektorienteret programmeringssprog udviklet af Microsoft som en del af .NET-platformen.

---

##### Nøglepunkter om (C#)##

- **Udviklet af Microsoft** og introduceret i starten af 2000’erne.
    
- En del af **.NET-økosystemet**, som gør det muligt at bygge applikationer til Windows, web, mobil, spil og meget mere.
    
- Stærkt typet og **objektorienteret** sprog med fokus på sikkerhed, ydeevne og robusthed.
    
- Understøtter moderne programmeringsparadigmer som **asynkron programmering**, **LINQ** (data queries), **lambda-udtryk**, **generics** og meget mere.
    
- Kan bruges til at udvikle mange typer software, bl.a.:
    
    - Desktop-applikationer (WPF, WinForms)
        
    - Web-applikationer og services (ASP.NET Core)
        
    - Mobilapps (Xamarin, MAUI)
        
    - Spiludvikling (Unity)
        
    - Cloud-løsninger (Azure)
        

---

##### Syntax og struktur

- C#’s syntaks minder meget om **C, C++ og Java**, så det er ret let for udviklere med erfaring i disse sprog at lære det.
    
- Kode organiseres i **klasser** og **metoder**.
    

Eksempel på simpel C#-kode:

```csharp
using System;

class Program
{
    static void Main()
    {
        Console.WriteLine("Hej verden!");
    }
}
```

---

## Hvorfor vælge C#?
##### 
- Stor community og mange ressourcer.
    
- Integreret med Microsofts værktøjer (Visual Studio).
    
- Tværplatform — kan køre på Windows, Linux og macOS med .NET Core/.NET 5+.
    
- Velegnet til både små og store projekter.
    
- Moderne features og konstant udvikling.
