## Forståelse af Binære, Hexadecimale, Decimale (Base-10) og andre talsystemer

Talsystemernes grundlag:

Decimal (Base-10)

- Cifre: 0-9 (10 forskellige cifre)
    
- Positionsværdi: Hver position er en potens af 10
    
- Eksempel: 1234₁₀ = (1×10³) + (2×10²) + (3×10¹) + (4×10⁰)
    
- Anvendelse: Hverdagstal, menneskeligt system
    

Binær (Base-2)

- Cifre: 0-1 (2 forskellige cifre)
    
- Positionsværdi: Hver position er en potens af 2
    
- Eksempel: 1101₂ = (1×2³) + (1×2²) + (0×2¹) + (1×2⁰) = 13₁₀
    
- Anvendelse: Computerens grundsprog (bits)
    

Hexadecimal (Base-16)

- Cifre: 0-9 og A-F (16 forskellige symboler)
    
- A=10, B=11, C=12, D=13, E=14, F=15
    
- Positionsværdi: Hver position er en potens af 16
    
- Eksempel: 2AF₁₆ = (2×16²) + (10×16¹) + (15×16⁰) = 687₁₀
    
- Anvendelse: Programmering, hukommelsesadresser, farver
    

Konverteringseksempler:

Decimal til Binær:

125₁₀ → 1111101₂

- 125 ÷ 2 = 62 rest 1
    
- 62 ÷ 2 = 31 rest 0
    
- 31 ÷ 2 = 15 rest 1
    
- (fortsæt og læs rester bagfra)
    

Binær til Hexadecimal:

11010110₂ → D6₁₆

- Grup i 4-bit: 1101 0110
    
- 1101₂ = 13₁₀ = D₁₆
    
- 0110₂ = 6₁₀ = 6₁₆
    

Andre talsystemer:

Octal (Base-8)

- Cifre: 0-7
    
- Anvendelse: Unix-filrettigheder
    
- Eksempel: 755₈ = rwxr-xr-x
    

Base-64

- Symboler: A-Z, a-z, 0-9, +, /
    
- Anvendelse: Email-vedlæg, web-kodning
    

Praktiske anvendelser i IT:

Binær:

- CPU-instruktioner
    
- Netværksmasker (255.255.255.0 = 11111111.11111111.11111111.00000000)
    

Hexadecimal:

- MAC-adresser (00:1A:2B:3C:4D:5E)
    
- IP-adresser i IPv6
    
- HTML-farver (#FF0000 = rød)
    
- Hukommelsesdumps
    

Hurtige konverteringer:

- 1 hex-ciffer = 4 binære bits
    
- 8 bits = 1 byte = 0-255₁₀ = 00-FF₁₆
